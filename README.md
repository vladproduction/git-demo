# git-demo

**BASIC GIT COMMANDS**:
 - git config --global user.name
 - git config --global user.email
 - git clone [link for remote repo]
 - git remote -v
 - git status
 - git add [file name]
 - git add .
 - git commit -m "commit message"
 - git log
 - git push
 - git pull
 - git gui&
 - gitk&
 - git restore . 
 - git restore [file name]
 - git clean -xdf
 - git reset HEAD~1
 - git reset --soft HEAD~1
 - git reset --mixed HEAD^ (HEAD^ same as HEAD~1)
 - git reset --hard HEAD^
